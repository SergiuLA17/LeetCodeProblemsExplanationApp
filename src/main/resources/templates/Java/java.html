Java ca crossplatforma

Java se porneste pe Java Virtual Machine. Noi scriem cod, care este transformat in bite cod. Realizara JMV este aproape pe toate sitemele de operare. Si Java scrisa o data poate fi folosita pe orce dispozititv. << Sloganul limbajului.ere

Nu este 100% bazata OOP

Are date primitive care nu sunt obiecte si se salveaza in stack-e si nu este nevoie de a le scaotem. Sunt bune pentru optimizare. Dar si ele au clase care prezinta adaptarea acestor primitive la obiecte, ele sunt numite wrapper class, aceste clase sunt create pentru ca in collectii nu este posibilitatea de a lucra cu primitive. Si aceste wrapper classe le inlocueste.
