Primitive

Un bit este o unitate de informatie, care poate avea doar doua valori: 0 sau 1.

In Java exista 8 tipuri de variabile primitive, care sunt: byte, short, int, long, float, double, boolean si char.

Un byte este o unitate de informatie, care poate avea 256 valori diferite. De la -128 la 127.

Un short este o unitate de informatie, care poate avea 65536 valori diferite. De la -32768 la 32767. Se folosesc mai des cand este nevoie de salvat memorie in lucrul cu tablouri mari

Un int este o unitate de informatie, care poate avea 2^32 valori diferite. De la -2147483648 la 2147483647.

Un long este o unitate de informatie, care poate avea 2^64 valori diferite. De la -9223372036854775808 la 9223372036854775807.

Un float este o unitate de informatie, care poate avea 2^32 valori diferite. De la -3.4028235E38 la 3.4028235E38. Are o precizie de 7 cifre dupa virgula.

Un double este o unitate de informatie, care poate avea 2^64 valori diferite. De la -1.7976931348623157E308 la 1.7976931348623157E308. Are o precizie de 16 cifre dupa virgula.

Un boolean este o unitate de informatie, care poate avea doar doua valori: true sau false.

Un char este o unitate de informatie, care poate avea 2^16 valori diferite. De la 0 la 65535.

Diferenta dinte float si double este ca float are o precizie mai mica, dar are o memorie mai mica.

Fiecare variabila primitiva are o clasa ambalaj(wrapper). Ele sunt create din cauza ca cu colectiile nu pot lucra cu primitivele.
